### Lessons Learned
- Cloudflare nameservers MUST be changed at the domain REGISTRAR (Hostinger), not just in Cloudflare dashboard
- Cloudflare Flexible SSL: visitor→cloudflare=HTTPS, cloudflare→origin=HTTP → works
- Cloudflare Full SSL: cloudflare→origin requires valid SSL on origin VPS → fails due to TLS cipher mismatch
- DNS Parking nameservers (ns1/ns2.dns-parking.com) must be replaced with Cloudflare nameservers at registrar
- Self-signed SSL cert on VPS causes ERR_SSL_VERSION_OR_CIPHER_MISMATCH with Cloudflare Full mode
- Next.js standalone output needs static files symlinked separately
