📰 How to Get Binance API Key
1. Log in to Binance.com
2. Go to Profile Icon → API Management
3. Click Create API
4. Choose System Generated
5. Name your key (e.g. "ZummpTrade Bot")
6. Enable: Read + Trading
7. DO NOT enable: Withdrawals
8. Complete security verification
9. Copy API Key and Secret Key and paste above
